usage
This site uses CSS custom properties for theming. All tokens are defined in :root (light) and .dark (dark mode). Components use var(--primary), var(--card), var(--border), etc. Border radius is 12px. Shadows are multi-layered and subtle. Mobile breakpoint at 600px. Container max-width 800px.
